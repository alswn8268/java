1. getElementById(): 태그에 지정된 id 속성을 탐색한다.
   같은 화면에 id 요소가 여러 개 있다면 첫 번째 요소 1개만 선택된다.

2. getElementsByName(): 태그에 지정된 name 속성을 탐색한다.
   같은 화면에 같은 name 속성을 가지는 요소가 여러 개 있다면 모두 선택된다. => NodeList
   
   
   

3. getElementsByTagName(): 태그 이름을 탐색한다.
   같은 화면에 같은 이름의 태그가 있다면 모두 선택된다. => NodeList

4. getElementsByClassName(): 태그에 지정된 class 속성을 탐색한다.
   같은 화면에 같은 class 속성을 가지는 요소가 모두 선택된다. => NodeList
